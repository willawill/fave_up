# Fave up

## This is an app built on top of Meetup for meetup users to save their favorite groups.

### Plans

- Flow type
- Jest testing
- Backend with cloudfire
- Api in node

### Data schema:

- User

	- ID: integer

- Favorite

	- Meetup ID: integer
	- User ID: integer
	- Added at: date time
	- has new event? boolean
